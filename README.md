## Lokale installatie

1. Clone de repository  
   ```bash
   git clone [repo-url]
   ```

2. Installeer dependencies  
   ```bash
   npm install
   ```

3. Start de development server
   ```bash
    ng serve
   ```

Open http://localhost:4200 in je browser
